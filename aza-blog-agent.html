✦ First-Time Setup

This app reads feed.json generated daily by GitHub Actions. Complete the steps below once:

1Push all files from this folder to a new GitHub repository
2Go to Settings → Pages → enable from main branch. Your app will be live at yourusername.github.io/aza-blog-agent
3Go to Actions → Daily Feed Refresh → Run workflow to generate the first feed.json
4After this, the feed auto-refreshes at 7:30 AM IST every day. Bookmark the URL and share with your team.
